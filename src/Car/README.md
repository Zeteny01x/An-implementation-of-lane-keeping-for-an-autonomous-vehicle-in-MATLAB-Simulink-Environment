## Car subsystem

The Car subsystem receives two input signals:

* **DriverCycle\[km/h]** – desired vehicle speed
* **SteeringAngleIn\[rad]** – steering angle

Both signals are generated by the external Controller block.

### Internal blocks

|Block|Inputs|Outputs|Description|
|-|-|-|-|
|**Driver**|DriverCycle\[km/h], SteeringAngleIn\[rad], VehicleSpeed\[km/h]|AccelerationPedal\[%], BrakePedal\[%], SteeringAngles\[rad]|Processes pedal and steering signals with speed feedback|
|**Engine**|AccelerationPedal\[%], EngineSpeed\[rad/s]|EngineTorque\[Nm]|Generates engine torque based on throttle position|
|**Driveline**|BrakePedal\[%], AccelerationPedal\[%], InputTorque\[Nm], VehicleSpeed\[km/h], Wheels|AxleTorque\[Nm], EngineSpeed\[rad/s]|Distributes torque to axles, provides speed feedback|
|**Wheels \& Brakes**|BrakePedal\[%], AxleTorque\[Nm], SteeringAngles\[rad], VehicleFeedback|FrontAxleForces\[N], RearAxleForces\[N], Wheels|Calculates wheel forces, provides wheel bus data|
|**Vehicle Dynamics**|FrontAxleForces\[N], RearAxleForces\[N]|VehicleFeedback, VehicleSpeed\[km/h], X\[m], Y\[m], Xdot\[m/s], Ydot\[m/s], Yaw\[deg], YawRate\[deg/s]|Calculates vehicle motion and position|

### Data flow

1. **Driver** block uses speed reference and feedback to generate pedal and steering outputs
2. **Engine** converts throttle position to torque
3. **Driveline** distributes torque to axles and provides engine speed feedback
4. **Wheels \& Brakes** calculates wheel forces using axle torque, steering angle, and brake input
5. **Vehicle Dynamics** computes vehicle position, speed, orientation, and motion states

### Outputs

The Car subsystem provides:

* Vehicle position: **X\[m]**, **Y\[m]**
* Vehicle velocity: **Xdot\[m/s]**, **Ydot\[m/s]**
* Speed: **VehicleSpeed\[km/h]**
* Orientation: **Yaw\[deg]**
* Yaw rate: **YawRate\[deg/s]**
